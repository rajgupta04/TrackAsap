# TrackAsap Mobile Assets

Place all static mobile assets here:
- `icon.png` (App Icon - 1024x1024)
- `splash.png` (Splash screen logo)
- `adaptive-icon.png` (Android Adaptive Icon foreground)
- `favicon.png` (Web favicon)

When running locally in Expo Go without physical images present, default Expo placeholders or fallback styles will render automatically.
